
# GutBrain Explorer

An interactive web app for exploring gut microbiome differences across diet groups using real Human Microbiome Project data.

## 🔗 Live App
[Click here to open GutBrain Explorer](https://sahas-13-gutbrain-explorer-app-uhp2kb.streamlit.app)

## Features
- Interactive heatmap of OTU abundance
- Volcano plot with statistical testing
- Top differentially abundant OTUs
- Shannon diversity analysis

## Built With
Python | Streamlit | Pandas | Scipy | Seaborn | Matplotlib

## Data
Human Microbiome Project (Turnbaugh et al.)
Processed OTU table with metadata — 675 samples, 6696 OTUs
